Hugoのコマンド3選

Jun 14, 2016   #hugo 

普段使う（というよりもこれ以外使ってない）コマンド3選

  • 記事追加
hugo new post/yyyy-mm-dd_title.md

年月日は好みで入れてます。

  • サーバー起動
hugo server -t angels-ladder -D -w

-D ・・・ ドラフトも起動する
-W ・・・ 保存時にページをリロード
angels-ladder ・・・ layout名

  • デプロイ
hugo -t angels-ladder
