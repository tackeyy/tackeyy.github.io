git commitを取り消す

Jul 21, 2016   #git  #tips 

間違えてdevelopにコミットしてもーた

そんな時にいつもコミットの取り消し方をググるので備忘がてらメモ。

コミットを取り消してワークディレクトリはそのままにする

  git reset --soft HEAD^

コミットを取り消してワークディレクトリの内容も書き換える

  git reset --hard HEAD^
